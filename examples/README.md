# Catalog Pipeline

This directory contains Python script examples demonstrating how to generate intake-ESM catalogs for various datasets. These scripts serve as practical guides for creating intake catalogs using the tools provided in this repository.

## Overview

The examples show real-world usage patterns for the `create_catalog.py` tool and related utilities. Each script is tailored for specific datasets and demonstrates different configuration options and data access patterns.

## Available Scripts

- [`d640000.jra3q.py`](https://github.com/NCAR/gdex-intake-esm/blob/main/examples/d640000.jra3q.py) : script to create the JRA3Q catalog that contain posix (GLADE access), https (https URL access), osdf (open science data federation access)
